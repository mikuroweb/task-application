# ⚔️ 日々の生活管理アプリ

> 「空いた時間がもったいないですからね。後方支援は任せてください！」

堀川国広（刀剣乱舞）と一緒に、毎日の生活を管理するパーソナルアプリです。

---

## 📱 機能一覧

### 🏠 ホーム
今日の日付と、時間帯に合わせた堀川くんからの一言がティッカー形式で流れます。
カレンダー・ToDo・家計簿・日記・買い物リストのサマリーを一目で確認できます。

### 📅 カレンダー
月ごとの予定を管理できます。予定のある日にはドットが表示されます。

### ✅ ToDo リスト
やることを追加・完了チェックできます。
達成率がプログレスバーで表示され、全部終わると堀川くんが褒めてくれます。

### 💴 家計簿
収入・支出をカテゴリ別に記録できます。
今月の収支バランスをひと目で確認できます。

### 📖 日記
気分を選んで日記を書くと、**堀川くんがAIで返事をしてくれます**。
内容をちゃんと読んだうえで、堀川くんらしい言葉をかけてくれます。

### 🛒 買い物リスト
カテゴリ別にアイテムを追加・個数管理できます。
カート済みチェックで購入済みアイテムを管理、まとめて削除も可能です。

---

## 🚀 使い方

GitHub Pages でホスティングしているため、URLにアクセスするだけで使えます。

**iPhoneでホーム画面に追加する方法：**

1. SafariでページのURLを開く
2. 下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 完了 🎉

---

## 🛠️ 技術情報

- 素のHTML / CSS / JavaScript（フレームワーク不使用）
- データはブラウザの `localStorage` に保存（サーバー不要）
- 日記の返信機能は [Anthropic API](https://www.anthropic.com/) を使用
